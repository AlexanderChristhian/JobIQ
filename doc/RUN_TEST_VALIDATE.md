# Run, Test, and Validate

## Prereqs
- Node.js 18 or newer.
- pnpm installed.

## Install
```bash
pnpm install
```

## Run (dev)
```bash
pnpm dev
```
Open http://localhost:3000

## Build
```bash
pnpm build
```

## Start (prod)
```bash
pnpm start
```

## Lint
```bash
pnpm lint
```

## Tests
No automated tests yet.

## Validation checklist
- Compare each route to Frontend_Reference for visual parity.
- Verify mock API routes return JSON without errors.
- Check keyboard navigation and visible focus states.
- Validate contrast for text, badges, and indicators.
