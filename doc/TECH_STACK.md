# JobIQ Tech Stack

## Frontend

- Next.js (App Router) with React and TypeScript.
- Tailwind CSS for styling and design tokens.
- Google Fonts: Manrope and Space Grotesk.
- Material Symbols Outlined icons.

## Server (WoZ)

- Next.js API routes for mocked data.
- Static JSON fixtures for recommendations, profile, and tracker data.

## Tooling

- pnpm for package management.
- ESLint (Next.js default config).

## Out of scope for WoZ

- No FastAPI or separate backend service.
- No database or persistent user storage.
- No real scraping or LLM inference.
